## What is ZBar?

ZBar is a comprehensive barcode scanning library that provides robust barcode detection and decoding capabilities across multiple platforms. Originally developed in 2007, the library continues to be actively maintained with support for modern video capture APIs and contemporary barcode formats.

The library supports a wide range of 1D and 2D barcode symbologies including EAN-13/UPC-A, UPC-E, EAN-8, Code 128, Code 93, Code 39, Codabar, Interleaved 2 of 5, QR Code, and SQ Code.

## Quick Example

Here's a simple example of scanning an image file using the C API:

```c scan_image.c lines
#include <stdio.h>
#include <zbar.h>

int main(int argc, char **argv)
{
    zbar_image_scanner_t *scanner;
    zbar_image_t *image;
    int n;

    /* create a reader */
    scanner = zbar_image_scanner_create();

    /* configure the reader */
    zbar_image_scanner_set_config(scanner, 0, ZBAR_CFG_ENABLE, 1);

    /* obtain image data */
    image = zbar_image_create();
    zbar_image_set_format(image, zbar_fourcc('Y','8','0','0'));
    zbar_image_set_size(image, width, height);
    zbar_image_set_data(image, raw_data, width * height, NULL);

    /* scan the image for barcodes */
    n = zbar_scan_image(scanner, image);

    /* extract results */
    const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
    for(; symbol; symbol = zbar_symbol_next(symbol)) {
        zbar_symbol_type_t typ = zbar_symbol_get_type(symbol);
        const char *data = zbar_symbol_get_data(symbol);
        printf("decoded %s symbol \"%s\"\n",
               zbar_get_symbol_name(typ), data);
    }

    /* clean up */
    zbar_image_destroy(image);
    zbar_image_scanner_destroy(scanner);

    return 0;
}
```

## Use Cases

ZBar is ideal for:

- **Retail Applications**: Point of sale systems, inventory management, and price checking
- **Document Processing**: Automated document routing and tracking
- **Access Control**: Badge and ticket scanning systems
- **Mobile Applications**: QR code readers and product information lookups
- **Logistics**: Package tracking and warehouse management
- **Healthcare**: Patient identification and medication tracking

## Architecture

The ZBar library is structured in layers:

**High-Level Interfaces**
- Processor: Complete scanning solution with display window
- GTK+ and Qt Widgets: GUI integration for applications

**Mid-Level Components**
- ImageScanner: Barcode detection in images
- Video: Video device capture abstraction
- Window: Display output abstraction

**Low-Level Components**
- Scanner: Intensity sample stream processing
- Decoder: Bar and space width decoding
