## Overview

The ImageScanner is a mid-level interface that reads barcodes from 2-D images. It provides the core scanning functionality used by higher-level interfaces like the Processor.

## Object Management

### zbar_image_scanner_create

Create a new image scanner instance.

```c
zbar_image_scanner_t *zbar_image_scanner_create(void);
```

<ResponseField name="return" type="zbar_image_scanner_t*">
  Pointer to the new image scanner object
</ResponseField>

**Example:**
```c
zbar_image_scanner_t *scanner = zbar_image_scanner_create();
```

### zbar_image_scanner_destroy

Destroy an image scanner and free all associated resources.

```c
void zbar_image_scanner_destroy(zbar_image_scanner_t *scanner);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner to destroy
</ParamField>

## Configuration

### zbar_image_scanner_set_config

Set configuration for specified symbology.

```c
int zbar_image_scanner_set_config(zbar_image_scanner_t *scanner,
                                   zbar_symbol_type_t symbology,
                                   zbar_config_t config,
                                   int value);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="symbology" type="zbar_symbol_type_t" required>
  The barcode symbology to configure (or 0 for all)
</ParamField>

<ParamField path="config" type="zbar_config_t" required>
  The configuration parameter to set
</ParamField>

<ParamField path="value" type="int" required>
  The value to set
</ParamField>

<ResponseField name="return" type="int">
  Returns 0 for success, non-0 for failure (config does not apply or value out of range)
</ResponseField>

**Common configurations:**

- `ZBAR_CFG_ENABLE` - Enable/disable symbology (1/0)
- `ZBAR_CFG_MIN_LEN` - Minimum data length for valid decode
- `ZBAR_CFG_MAX_LEN` - Maximum data length for valid decode
- `ZBAR_CFG_POSITION` - Enable position data collection
- `ZBAR_CFG_X_DENSITY` - Vertical scan density
- `ZBAR_CFG_Y_DENSITY` - Horizontal scan density

**Example:**
```c
// Enable QR codes only
zbar_image_scanner_set_config(scanner, 0, ZBAR_CFG_ENABLE, 0);
zbar_image_scanner_set_config(scanner, ZBAR_QRCODE, ZBAR_CFG_ENABLE, 1);

// Set minimum length for Code 128
zbar_image_scanner_set_config(scanner, ZBAR_CODE128, ZBAR_CFG_MIN_LEN, 6);
```

### zbar_image_scanner_get_config

Retrieve configuration for specified symbology.

```c
int zbar_image_scanner_get_config(zbar_image_scanner_t *scanner,
                                   zbar_symbol_type_t symbology,
                                   zbar_config_t config,
                                   int *value);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="symbology" type="zbar_symbol_type_t" required>
  The barcode symbology to query
</ParamField>

<ParamField path="config" type="zbar_config_t" required>
  The configuration parameter to retrieve
</ParamField>

<ParamField path="value" type="int*" required>
  Pointer to store the retrieved value
</ParamField>

<ResponseField name="return" type="int">
  Returns 0 for success, non-0 for failure. On success, *value is filled with the configuration value.
</ResponseField>

### zbar_image_scanner_parse_config

Parse and apply configuration from a string.

```c
int zbar_image_scanner_parse_config(zbar_image_scanner_t *scanner,
                                     const char *config_string);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="config_string" type="const char*" required>
  Configuration string in format "[symbology.]config[=value]"
</ParamField>

<ResponseField name="return" type="int">
  Returns 0 for success, non-0 for failure
</ResponseField>

**Example:**
```c
zbar_image_scanner_parse_config(scanner, "qrcode.enable=1");
zbar_image_scanner_parse_config(scanner, "code128.min-length=6");
```

## Scanning

### zbar_scan_image

Scan an image for barcodes.

```c
int zbar_scan_image(zbar_image_scanner_t *scanner,
                    zbar_image_t *image);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="image" type="zbar_image_t*" required>
  The image to scan (must be in Y800 or GRAY format)
</ParamField>

<ResponseField name="return" type="int">
  - Positive: Number of symbols successfully decoded
  - 0: No symbols found
  - Negative: Error occurred
</ResponseField>

<Warning>
The image format must be "Y800" or "GRAY". Use `zbar_image_convert()` to convert from other formats.
</Warning>

**Example:**
```c
// Create and configure scanner
zbar_image_scanner_t *scanner = zbar_image_scanner_create();
zbar_image_scanner_set_config(scanner, 0, ZBAR_CFG_ENABLE, 1);

// Create and setup image
zbar_image_t *image = zbar_image_create();
zbar_image_set_format(image, zbar_fourcc('Y', '8', '0', '0'));
zbar_image_set_size(image, width, height);
zbar_image_set_data(image, raw_data, width * height, zbar_image_free_data);

// Scan for barcodes
int n = zbar_scan_image(scanner, image);
if (n < 0) {
    fprintf(stderr, "Scan error\n");
} else {
    printf("Found %d symbols\n", n);
}

// Extract results
const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
for (; symbol; symbol = zbar_symbol_next(symbol)) {
    const char *data = zbar_symbol_get_data(symbol);
    printf("Decoded: %s\n", data);
}

// Cleanup
zbar_image_destroy(image);
zbar_image_scanner_destroy(scanner);
```

## Caching

### zbar_image_scanner_enable_cache

Enable or disable the inter-image result cache.

```c
void zbar_image_scanner_enable_cache(zbar_image_scanner_t *scanner,
                                      int enable);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="enable" type="int" required>
  1 to enable caching, 0 to disable
</ParamField>

The cache is useful for scanning video frames. It:
- Filters duplicate results from consecutive images
- Adds consistency checking to results
- Provides hysteresis to reduce noise
- Clears when this function is called

<Note>
The cache is disabled by default. Enable it when processing video streams to reduce duplicate detections.
</Note>

**Example:**
```c
// Enable cache for video stream processing
zbar_image_scanner_enable_cache(scanner, 1);

// Process video frames
while (capturing) {
    zbar_image_t *frame = get_next_frame();
    zbar_scan_image(scanner, frame);
    // Cache will filter out duplicate results
}
```

### zbar_image_scanner_recycle_image

Remove decoded results from an image and scanner.

```c
void zbar_image_scanner_recycle_image(zbar_image_scanner_t *scanner,
                                       zbar_image_t *image);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="image" type="zbar_image_t*" required>
  The image to recycle
</ParamField>

This is a more efficient version of `zbar_image_set_symbols(image, NULL)` that retains memory for subsequent decodes.

## Results

### zbar_image_scanner_get_results

Retrieve decode results for the last scanned image.

```c
const zbar_symbol_set_t *zbar_image_scanner_get_results(
    const zbar_image_scanner_t *scanner);
```

<ParamField path="scanner" type="const zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ResponseField name="return" type="const zbar_symbol_set_t*">
  The symbol set result container, or NULL if no results available
</ResponseField>

<Warning>
The symbol set does not have its reference count adjusted. Increment the count if you need to keep results after the next scan.
</Warning>

## Callbacks

### zbar_image_scanner_set_data_handler

Setup result handler callback.

```c
zbar_image_data_handler_t *zbar_image_scanner_set_data_handler(
    zbar_image_scanner_t *scanner,
    zbar_image_data_handler_t *handler,
    const void *userdata);
```

<ParamField path="scanner" type="zbar_image_scanner_t*" required>
  The image scanner instance
</ParamField>

<ParamField path="handler" type="zbar_image_data_handler_t*" required>
  Callback function (or NULL to disable)
</ParamField>

<ParamField path="userdata" type="const void*">
  User data passed to callback
</ParamField>

<ResponseField name="return" type="zbar_image_data_handler_t*">
  The previously registered handler
</ResponseField>

The callback function signature:
```c
typedef void(zbar_image_data_handler_t)(zbar_image_t *image,
                                        const void *userdata);
```

**Example:**
```c
void my_handler(zbar_image_t *image, const void *userdata) {
    const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
    for (; symbol; symbol = zbar_symbol_next(symbol)) {
        printf("Decoded: %s\n", zbar_symbol_get_data(symbol));
    }
}

zbar_image_scanner_set_data_handler(scanner, my_handler, NULL);
```
