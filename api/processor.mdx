## Overview

The Processor is a high-level interface that processes video and images for barcodes, optionally displaying them in a library-owned output window. It integrates video capture, image scanning, and display into a single convenient interface.

## Key Features

- Integrated video capture and scanning
- Optional display window with overlay
- Threading support for improved responsiveness
- Built-in result callbacks
- Video device control

## Object Management

### zbar_processor_create

Create a new processor instance.

```c
zbar_processor_t *zbar_processor_create(int threaded);
```

<ParamField path="threaded" type="int" required>
  If non-zero and threading is available, the processor will spawn threads to avoid blocking and improve responsiveness
</ParamField>

<ResponseField name="return" type="zbar_processor_t*">
  Pointer to the new processor object
</ResponseField>

**Example:**
```c
// Create with threading enabled
zbar_processor_t *proc = zbar_processor_create(1);
```

### zbar_processor_destroy

Destroy a processor and clean up all associated resources.

```c
void zbar_processor_destroy(zbar_processor_t *processor);
```

<ParamField path="processor" type="zbar_processor_t*" required>
  The processor to destroy
</ParamField>

## Initialization

### zbar_processor_init

Initialize the processor with video input and/or display output.

```c
int zbar_processor_init(zbar_processor_t *processor,
                        const char *video_device,
                        int enable_display);
```

<ParamField path="processor" type="zbar_processor_t*" required>
  The processor instance
</ParamField>

<ParamField path="video_device" type="const char*">
  Platform-specific video device name (e.g., "/dev/video0" on Linux), or NULL for no video input
</ParamField>

<ParamField path="enable_display" type="int" required>
  Non-zero to enable the display window
</ParamField>

<ResponseField name="return" type="int">
  Returns 0 on success, -1 on error
</ResponseField>

**Example:**
```c
zbar_processor_t *proc = zbar_processor_create(1);

// Initialize with video device and display
if (zbar_processor_init(proc, "/dev/video0", 1) < 0) {
    fprintf(stderr, "Failed to initialize: %s\n",
            zbar_processor_error_string(proc, 0));
    zbar_processor_destroy(proc);
    return -1;
}
```

## Processing Control

### zbar_processor_set_active

Control the processor in free running video mode.

```c
int zbar_processor_set_active(zbar_processor_t *processor,
                              int active);
```

<ParamField path="processor" type="zbar_processor_t*" required>
  The processor instance
</ParamField>

<ParamField path="active" type="int" required>
  Non-zero to start processing, 0 to stop
</ParamField>

<ResponseField name="return" type="int">
  Returns 0 on success, -1 on error
</ResponseField>

Only works if video input is initialized. If threading is enabled, scanning occurs in the background. If the display window is visible, video display will be enabled.

### zbar_process_one

Process video until a result is available or timeout expires.

```c
int zbar_process_one(zbar_processor_t *processor,
                     int timeout);
```

<ParamField path="processor" type="zbar_processor_t*" required>
  The processor instance
</ParamField>

<ParamField path="timeout" type="int" required>
  Timeout in milliseconds (-1 for indefinite)
</ParamField>

<ResponseField name="return" type="int">
  - Positive: Symbols successfully decoded
  - 0: No symbols found (timeout expired)
  - -1: Error occurred
</ResponseField>

If the display window is visible, video will be shown. Despite the name, multiple results may be returned.

**Example:**
```c
// Scan until barcode found or user aborts
int n = zbar_process_one(proc, -1);
if (n > 0) {
    printf("Found %d symbols\n", n);
}
```

## Callbacks

### zbar_processor_set_data_handler

Setup result handler callback.

```c
zbar_image_data_handler_t *zbar_processor_set_data_handler(
    zbar_processor_t *processor,
    zbar_image_data_handler_t *handler,
    const void *userdata);
```

<ParamField path="processor" type="zbar_processor_t*" required>
  The processor instance
</ParamField>

<ParamField path="handler" type="zbar_image_data_handler_t*" required>
  Callback function (or NULL to disable)
</ParamField>

<ParamField path="userdata" type="const void*">
  User data passed to callback
</ParamField>

<ResponseField name="return" type="zbar_image_data_handler_t*">
  The previously registered handler
</ResponseField>

The callback is invoked whenever new results are available from the video stream or static image.

**Example:**
```c
static void my_handler(zbar_image_t *image, const void *userdata) {
    const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
    for (; symbol; symbol = zbar_symbol_next(symbol)) {
        zbar_symbol_type_t type = zbar_symbol_get_type(symbol);
        const char *data = zbar_symbol_get_data(symbol);
        printf("decoded %s symbol \"%s\"\n",
               zbar_get_symbol_name(type), data);
    }
}

zbar_processor_set_data_handler(proc, my_handler, NULL);
```

## Complete Example

```c
#include <stdio.h>
#include <zbar.h>

static void my_handler(zbar_image_t *image, const void *userdata) {
    const zbar_symbol_t *symbol = zbar_image_first_symbol(image);
    for (; symbol; symbol = zbar_symbol_next(symbol)) {
        zbar_symbol_type_t typ = zbar_symbol_get_type(symbol);
        const char *data = zbar_symbol_get_data(symbol);
        printf("decoded %s symbol \"%s\"\n",
               zbar_get_symbol_name(typ), data);
    }
}

int main(int argc, char **argv) {
    const char *device = "/dev/video0";

    // Create processor with threading
    zbar_processor_t *proc = zbar_processor_create(1);

    // Enable all symbologies
    zbar_processor_set_config(proc, 0, ZBAR_CFG_ENABLE, 1);

    // Initialize with video device
    if (argc > 1)
        device = argv[1];
    zbar_processor_init(proc, device, 1);

    // Setup callback
    zbar_processor_set_data_handler(proc, my_handler, NULL);

    // Show window and start processing
    zbar_processor_set_visible(proc, 1);
    zbar_processor_set_active(proc, 1);

    // Wait for user input to exit
    zbar_processor_user_wait(proc, -1);

    // Cleanup
    zbar_processor_destroy(proc);

    return 0;
}
```
